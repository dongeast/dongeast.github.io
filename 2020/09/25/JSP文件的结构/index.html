JSP文件的结构

Dongeast 2020-09-25 11:53:53
软件理论

在传统的网页HTML（.htm、.html）中加入Java程序片段，就构成了JSP网页（.JSP）。

创建第一个JSP文件

<%@ page contentType="text/html;charset=GBK"%>
<html>
<head>
<title>我的第一个JSP程序！</title>
</head>
<body>
<%!int number = 1;%>
<%--这是声明一个变量--%>
<%!public int count(){
    return number++;
}
/*这是声明一个方法*/%>
<%
//JSP-code
out.println("Hello JSP!");
out.println("欢迎使用JSP交互式动态网页！！  ");
%>
<br>
<%="您是第"+count()+"个客人！">
<br>
</body>
</html>

这个实例的程序代码可以分为两个部分：HTML标签和Java程序代码。

JSP文件的组成元素

脚本元素
1.<%  %>之间的内容是Java程序片段：
    其他的是HTML标签，第一行为JSP指令元素。
    程序被编译后重新生成全HTML标签文字，重新建立只包含HTML的网页内容。

2.隐藏注释（不发送到客户端）：
    <%--  --%>

3.数据输出：
    out.println("  ");

4.表达式：
    <%="您是第"+count()+"个客人！">

    <%=  %>
    直接输出表达式的结果。
    不能用分号作为表达式的结尾。

5.声明：
    <%!int number = 1;%>
    可以直接使用在<%@ page %>中被包含进来的已经声明的变量和方法。
    一个声明仅在一个页面中有效。

6.HTML注释（显式注释/客户端可以看到）：
    <!--  -->  可在客户端网页的源码中显示

7.Java注释：

8.    <%  %> 局部变量
    <%!  %> 全局变量

9.脚本段：
    <%  %>直接放入到Servlet的方法内。
指令元素
1.page指令：
    <%@ 指令名 属性1 = “值1” 属性2 = “值2” ... %>
    JSP包括page、include、taglib 3种指令。
